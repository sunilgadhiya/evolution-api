# Integrations

## Messaging Providers
- **Baileys**: WhatsApp Web WebSocket implementation.
- **WhatsApp Cloud API**: Official Meta API.

## Chatbots & AI
- **Typebot**: Visual bot builder integration.
- **Chatwoot**: Omnichannel customer support inbox.
- **Dify**: LLM Agent workflows.
- **OpenAI**: Used for Whisper audio transcription and AI responses.

## Event Buses
- **RabbitMQ / SQS / Kafka / NATS**: External message queue systems for highly reliable event streaming in enterprise deployments.
- **Socket.io**: Real-time websocket event emission for web clients.

## Storage
- **MinIO / AWS S3**: Remote media storage for received WhatsApp files.
