# Business Context

## Product Vision & Mission
Evolution API is an open-source REST API engine for WhatsApp and multi-channel messaging. It abstracts the complexity of connecting to WhatsApp (via Baileys or Official Cloud API) and exposes a clean, unified REST interface for sending and receiving messages.

## Target Users & Personas
- **Developers**: Integrating WhatsApp into CRM, ERP, and support platforms.
- **Businesses**: Automating customer support, marketing, and notifications.

## Business Model
- Open Source (Apache 2.0) upstream project.
- Maintained by Evolution Foundation.

## Core Value Propositions
- Unifies multiple WhatsApp connection methods (Web/Baileys & Cloud).
- Built-in multi-tenancy (multiple WhatsApp instances on one server).
- Extensive ecosystem integrations (Typebot, Chatwoot, Dify, OpenAI).

## Current Status
- Production-ready, stable OSS project (v2.3.7).

## Key Success Metrics
- Message delivery latency.
- Connection stability (uptime of Baileys sessions).
- Webhook delivery reliability.
